# 灵界（LingVerse）游戏辅助脚本

适用于 [灵界](https://ling.muge.info/) 的 Tampermonkey 用户脚本集合，提升游戏体验。

## 📦 包含脚本

| 脚本 | 版本 | 功能 |
|------|------|------|
| **天道试炼塔自动挑战助手** | v4.3.2 | 自动挑战试炼塔，智能选择天赋 |
| **灵界储物助手** | v2.5 | 储物出售管理，批量出售物品 |

---

## ✨ 天道试炼塔自动挑战助手

### 功能特性
- ⚔️ **自动挑战**：自动循环挑战试炼塔，支持设置目标层数
- 🎯 **智能选天赋**：暴击优先排序，同类型词条自动选最大值，支持四种策略
- 💎 **灵石刷新**：自动使用灵石刷新天赋池，优先获取高品质天赋
- 📊 **天赋追踪**：实时显示天赋加成效果（攻击/防御/生命/灵力/暴击等）
- 🧬 **天赋组合记录**：记录当前轮次获得的所有天赋及稀有度统计
- 🧘 **冥想控制**：智能冥想管理，战斗中不自动冥想，战斗结束自动恢复
- 🌓 **主题切换**：日间/夜间模式，保护视力
- 🔑 **盐值验证**：自动获取会话盐值，未登录时脚本不激活，避免无效请求
- ⏳ **API退避机制**：API请求失败时自动递增等待时间（5秒~60秒），恢复后自动重置
- ⌨️ **快捷键**：`Ctrl+T` 快速开始/停止
- 📱 **双端适配**：PC端浮动面板 + 移动端抽屉式面板
- 💾 **位置记忆**：面板和按钮位置自动保存

### 天赋策略说明

| 策略 | 特点 |
|------|------|
| 综合平衡（暴击优先） | 优先暴击属性，兼顾各项数值 |
| 攻击优先 | 优先选择攻击类天赋 |
| 防御优先 | 优先选择防御/生命类天赋 |
| 传说品质优先 | 优先选择传说品质天赋 |

### 同词条选最大机制（v4.3.2 新增）
当同时出现多个相同类型的天赋词条时（如多个攻击类天赋），脚本会自动比较数值大小，优先选择数值最高的词条：
- 📊 **数值优先**：同类型词条中，选择加成数值最大的
- 🏆 **同数值选品质**：数值相同时，优先选择品质更高的
- 📋 **日志记录**：选择过程会在日志中显示"同类型N个词条，选最大"

### 特殊词条识别
自动识别并优先选择以下特殊天赋：
- ✨ **不死**：首次HP归零恢复20%HP
- ⚔️ **斩杀**：敌人HP<20%伤害翻倍
- 🌀 **天道汲取**：击杀回复30%HP+30%MP
- 🌿 **灵根共鸣**：灵根战斗特效强化
- ⚡ **天怒**：攻击+30%

### API退避机制说明
当脚本的API请求连续失败时，会自动进入退避状态：
- 第1次失败 → 等待 5 秒
- 第2次失败 → 等待 10 秒
- 第3次失败 → 等待 15 秒
- ...
- 最大等待上限 → 60 秒

一旦某次请求成功，退避计数器立即归零，恢复正常请求频率。

## 📦 灵界储物助手

### 功能特性
- 🔍 **物品搜索**：支持模糊搜索，快速定位物品
- 💰 **批量出售**：识别可售物品并一键出售
- 📊 **价值统计**：计算可售物品总价值
- 🏷️ **状态标记**：显示装备/绑定/锁定/冻结等状态
- 📱 **双端适配**：PC端浮动面板 + 移动端优化
- 💾 **位置记忆**：面板和按钮位置自动保存
- 🔄 **刷新同步**：与游戏储物数据实时同步

### 物品状态说明
- 🔒 **已锁定**：需要手动解锁后才能出售
- ⚔️ **已装备**：当前角色装备中
- 🌐 **已绑定**：已绑定无法交易
- ❄️ **冻结中**：交易冷却中，显示剩余时间
- ⚠️ **破损**：显示磨损率和折价后售价

## 📥 安装方法

### 前提条件

#### 桌面端浏览器
安装浏览器扩展 **Tampermonkey** 或 **Violentmonkey**：
- [Chrome 版 Tampermonkey](https://chrome.google.com/webstore/detail/tampermonkey/dhdgffkkebhmkfjojejmpbldmpobfkfo)
- [Firefox 版 Tampermonkey](https://addons.mozilla.org/en-US/firefox/addon/tampermonkey/)
- [Edge 版 Tampermonkey](https://microsoftedge.microsoft.com/addons/detail/tampermonkey/iikmkjmpaadaobahmlepeloendndfphd)
- [Violentmonkey](https://violentmonkey.github.io/)

#### Via 浏览器（Android）
1. 下载安装 [Via 浏览器](https://viayoo.com/)
2. 打开 Via 浏览器，点击底部菜单 → **设置** → **脚本**
3. 开启 **脚本功能** 开关
4. 脚本功能已激活，可导入用户脚本

### 安装脚本

#### 方式一：直接安装（推荐）
点击下方链接自动安装：

- [安装 天道试炼塔自动挑战助手](https://raw.giteeusercontent.com/qiyes/ling-verse-script/raw/master/ling-trial-tower-assistant.js)
- [安装 灵界储物助手](https://raw.githubusercontent.com/ming-a1/LingVerse-Script/refs/heads/main/lingverse-sell-helper.js)

#### 方式二：手动安装
1. 打开 Tampermonkey/Violentmonkey 管理面板
2. 点击 **创建新脚本**
3. 复制对应脚本文件内容并粘贴
4. 保存即可

#### 方式三：Via 浏览器安装
1. 打开 Via 浏览器，进入 **设置** → **脚本**
2. 点击右上角 **+** 号 → **从链接导入**
3. 粘贴脚本直链地址：
   - 天道试炼塔自动挑战助手：`https://raw.giteeusercontent.com/qiyes/ling-verse-script/raw/master/ling-trial-tower-assistant.js`
   - 灵界储物助手：`https://raw.githubusercontent.com/ming-a1/LingVerse-Script/refs/heads/main/lingverse-sell-helper.js`
4. 点击 **确定** 导入
5. 确保脚本开关已开启

> 💡 也可直接复制脚本全部代码，在 Via 脚本页面点击 **+** → **新建**，粘贴代码后保存。

## 🎮 使用方法

### 天道试炼塔自动挑战助手

1. 进入游戏后，打开 **天道试炼塔** 页面
2. 脚本会自动显示浮动按钮/面板
3. 配置策略和选项：
   - 选择天赋选择策略
   - 开启/关闭失败重试、跳过动画、灵石刷新等
   - 设置目标层数（0表示无限挑战）
4. 点击 **开始挑战** 或按 `Ctrl+T`
5. 实时查看天赋加成和战斗日志

### 灵界储物助手

1. 打开游戏中的 **储物面板**
2. 点击浮动按钮 📦 展开助手面板
3. 在搜索框输入物品名称（支持模糊搜索）
4. 按回车或点击搜索按钮
5. 查看物品列表，点击 **出售** 按钮出售对应物品

## 💡 常见问题

<details>
<summary><strong>Q: 脚本没有生效？</strong></summary>

- 确保已正确安装 Tampermonkey 扩展 / Via 脚本功能已开启
- 检查脚本是否处于启用状态
- 确认页面地址匹配 `ling.muge.info`
- **试炼塔助手需登录游戏后才能激活**（需获取会话盐值）
- 尝试刷新页面
</details>

<details>
<summary><strong>Q: 控制台显示"等待登录，API已退避"？</strong></summary>

- 这是正常提示，说明游戏尚未登录或盐值未获取
- 登录游戏后脚本会自动激活并正常工作
- 如果长时间未恢复，尝试刷新页面重新登录
</details>

<details>
<summary><strong>Q: 多个同类型天赋如何选择？</strong></summary>

- v4.3.2 起支持同词条选最大机制
- 脚本会自动比较同类型词条的数值大小
- 优先选择数值最大的，数值相同时选品质更高的
- 选择过程会在战斗日志中显示
</details>

<details>
<summary><strong>Q: 出售按钮点击无效？</strong></summary>

- 请先打开游戏原生储物面板以激活出售功能
- 确保物品未被锁定或绑定
</details>

<details>
<summary><strong>Q: 自动挑战停止不工作？</strong></summary>

- 检查网络连接是否正常
- 查看战斗日志了解具体原因
- 确认灵石是否充足（如果开启了刷新功能）
- API请求失败后会自动进入退避等待，请耐心等候
</details>
